### Proyecto actual  
**VUT (Vivienda de Uso Turístico)**  
Aplicación web full stack en desarrollo como **proyecto final de DAW**, orientada a la gestión de viviendas turísticas.  
Características:  
- Frontend, backend y base de datos integrados en un monolito Laravel.  
- Control de versiones con Git/GitHub.  
- **Estado actual**: en construcción, será desplegada en dominio propio como parte de mi portfolio profesional (uso demostrativo).  
- **Evolución prevista**: separación de frontend en Angular y posterior migración del backend a Spring Boot.  

Un proyecto diseñado para consolidar conocimientos, obtener la titulación de DAW y mostrar un producto real en funcionamiento.
